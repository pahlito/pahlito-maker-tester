# Pahlito Maker Tester

Proyectos base para generar servicios rest usando Spring Boot 

*QDSL, Swagger y Mock incluidos*

## Ejercicio:

######Se deben incluir las dependencias Maven y configuraciones de Spring Boot Starter de tal forma quese cumpla:

1.- Configurar el Parent para que el projecto sea Spring Boot

2.- Forzar el compilado en Java 1.8 en el Parent

3.- Agregar las dependencias para Spring Data JPA en el CORE
		
4.- Agregar la configuración para Entities y Repositories en el CORE

5.- Generar las clases QDSL en el CORE

6.- Agregar dependencias para MySQL en el CORE
		
7.- Agregar dependencias para Test JUnit (con Mock) en el CORE

8.- Agregar dependecias en la API para que pueda desplegar servicios web

9.- Agregar anotaciones para que la API sea una Spring Boot Aplication

10.- Crear la configuración para Spring Boot en la API de forma que reconozca los package de la aplicación
		
11.- Hacer que la API se genere como un proyecto Spring Boot Ejecutable

12.- Configurar conexión a la base de datos en las properties de la API

13.- Agregar Swagger UI a la API
